Timestamp Microservice API

User stories:

1) I can pass a string as a parameter, and it will check to see whether that string contains 
        either a unix timestamp or a natural language date (example: January 1, 2016)
2) If it does, it returns both the Unix timestamp and the natural language form of that date.
3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Usage:

https://aqueous-tor-13950.herokuapp.com/August 20, 2016
https://aqueous-tor-13950.herokuapp.com/1471701715

Example Response:

{
  normal: "August 20, 2016",
  unix: "1471701715"
}
